add cards in the HTML file

Open Index.html in a text editor. Search for MY_CARDS. You will find a clearly marked array near the top of the script section. Add entries like:

{q:'Your question here', a:'Your answer here'},

Optionally add code:'your_snippet()' to show a code block on the back. Save the file and upload it to GitHub.

how to add videos

YouTube: any link — youtube.com/watch?v=, youtu.be/, /shorts/. The ID is the 11-character string after v=.

TikTok: copy the full URL from the browser address bar. It must contain @user/video/NUMBER. Short vm.tiktok.com links will not work — open them in a browser first to get the full URL.

add videos directly in the HTML file

Open Index.html in a text editor. Search for ADD VIDEOS HERE. You will find a clearly labelled array. Copy any existing line and change four values:

{id:'yt_YOURID', type:'yt', vid:'YOURID', title:'Title', creator:'Name', long:false}

Set type:'tt' for TikTok and add handle:'@username'. Set long:true for long tutorials. Save and upload to GitHub.

install as app (PWA)

1. Upload both Index.html and icon.png to your GitHub repo.

2. Open your GitHub Pages URL in Chrome on Android.

3. Chrome menu → Add to Home screen. The icon.png file becomes your app icon.
