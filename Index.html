how to add videos

YouTube: paste any normal link: youtube.com/watch?v=..., youtu.be/... or youtube.com/shorts/.... Tap and hold speed control works on these.

TikTok: in the TikTok app use Share, then Copy link. Paste the full link that looks like tiktok.com/@user/video/123.... If you get a short vm.tiktok.com link, open it in a browser first and copy the full URL from the address bar. TikTok's embedded player keeps its own controls, so tap to pause and 2x hold do not apply to TikToks.

Longform: tick the box for long tutorials so they stay hidden until you switch the longform toggle on in the feed.

install as an app (PWA)

This is one HTML file. To use it as an installable app it needs to be served over HTTPS:

1. Put the file in a GitHub repo, enable GitHub Pages, open the URL.

2. On the fold: Chrome menu, "Add to Home screen". On desktop Chrome: install icon in the address bar.

Self hosting also works, e.g. an nginx container in your Docker stack or python -m http.server on the LAN. Saved playlists and added videos are stored per device in the browser.
